# 🎓 Digital Platform for Centralized Alumni Data Management and Engagement

## 📖 Project Overview

The Digital Platform for Centralized Alumni Data Management and Engagement is a web-based application developed to connect students, alumni, and administrators through a single platform. It enables alumni networking, job sharing, event management, and profile management.

---

## ✨ Features

- 👤 Student Login
- 🛡️ Admin Login
- 👨‍🎓 Alumni Management
- 🔍 Alumni Search
- 💼 Job Portal
- 📅 Event Management
- 📝 User Profile
- 🚪 Logout

---

- HTML5

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/Alumni-Management-System.git
```

2. Open the project folder.

3. Install the required packages.

```bash
pip install flask flask-mysqldb
```

4. Import the MySQL database.

5. Run the application.

```bash
python app.py
```

6. Open your browser and visit:

```
http://127.0.0.1:5000
```
